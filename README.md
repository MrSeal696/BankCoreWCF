# BankCoreWCF

Простой проект на **.NET 10 + CoreWCF**, реализующий систему управления банковскими счетами.

## Функционал

- **Клиенты** могут:
  - Просматривать свои счета (`GetAccounts`)
  - Просматривать транзакции по счетам (`GetTransactions`)
- **Операторы** могут:
  - Просматривать список всех клиентов (`GetAllClients`)
- **Администраторы** могут:
  - Блокировать счета (`BlockAccount`)
  - Разблокировать счета (`UnblockAccount`)
- Все операции могут быть защищены через роли (в будущем можно добавить авторизацию).

---

## Требования

- Visual Studio 2022 или 2023
- .NET 10 SDK
- Пакеты NuGet:
  - `CoreWCF`
  - `CoreWCF.Http`
